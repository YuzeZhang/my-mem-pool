通过剖析开源代码可以积累优秀的代码设计思想和良好的编程规范，了解不同应用场景下不同的内存池实现也是一种重要的能力，本仓库对SGI STL二级空间配置器源码和nginx内存池源码进行了剖析和注释，并将其分别移植成简单易用的形式。

本仓库包含以下内容：

1. 注释过的SGI STL二级空间配置器源码以及对其的分析整理
2. 注释过的nginx内存池源码以及对其的分析整理
3. my_allocator
4. my_nginx
